# react-template

## Developing

### Prerequisites

На компьютере должен быть установлен [Node.js](https://nodejs.org/en/).

### Setting up Dev

Установить все зависимости.

```shell
npm install
```

И запустить режим разработки.

```shell
npm start
```

Во вкладке браузера перейти по адресу
[http://localhost:3000](http://localhost:3000).
